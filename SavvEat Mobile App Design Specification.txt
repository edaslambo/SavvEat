Web Application Prompt Framework
1. User Registration and Authentication

    Implement secure user registration form with fields: email, password, display name, optional phone.

    Enable login/logout, password reset, and email verification features.

    Store user profile securely.

2. User Profile & Preference Input

    After registration, prompt user to fill out a guided “User Preferences” wizard:

        Personal info: Age, Gender, Height, Weight, BMI, activity level

        Dietary preferences: Vegetarian, vegan, low-carb, low-salt, etc.

        Food allergies/intolerances: List of common allergens (nuts, shellfish, gluten, etc.)

        Taste preferences: spice level, favorite cuisines (checkbox list: Chinese, Japanese, Western, etc.)

        Budget per meal/week

    Save profile for ongoing personalization.

3. Application Onboarding/Guidance

    On first login, display a short step-by-step guidance tour:

        Explain the app’s core value (personalized menu, cost optimization, health tracking)

        Illustrate main dashboard features: menu generation, shopping list, nutrition & cost visualization

        Show quick links to main actions (generate menu, update profile, track nutrition)

    Include a contextual help button at all steps.

4. AI-Powered Recipe Generation

    Main dashboard function: “Generate My Menu”

        Leverage user preferences and health data to recommend Chinese-style or user-selected recipes

        Provide controls for meal type (breakfast/lunch/dinner/snack) and portion

        Display list of suggested recipes with images, estimated prep time, serving size

    Option to regenerate/refine suggestions

5. Automated Budget Calculation

    For each generated recipe, automatically list all required ingredients, matching local grocery prices via API/crawling.

    Sum up the total estimated cost for meal/recipe/day/week.

    Display breakdown in a visually clear cost panel.

    Allow quick swapping of ingredients to see live cost changes.

6. Nutrition Value Calculation & Progress Tracking

    Analyze all recipes in the user’s daily menu:

        Show calories, key macronutrients (protein/fat/carbs), micronutrients (iron, calcium, vitamin C etc.)

        Use data visualization (bars, charts, progress rings) to display progress towards the user’s daily intake goals

        Grade or highlight recipes that most closely meet user’s nutrition requirements

    Option to view nutrition for each recipe, each day, and cumulative weekly intake.

Sample User Flow

    Register account → verify email/login

    Complete preferences & health profile wizard

    Receive guided tour of features

    Hit “Generate Menu” → review personalized recipes

    See per-meal and per-week grocery budget (with adjustable options)

    View nutrition content for each dish and track overall daily intake progress

Additional Guidance (Optional)

    Build with modular, reusable components for profile, input forms, recipe cards, budget, and nutrition panels

    Focus on seamless user experience: defaults, suggestions, inline validation, easy navigation

    Ensure mobile-responsive design and accessibility compliance
